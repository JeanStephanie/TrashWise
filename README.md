## Project Description
TrashWise is a web-based digital waste management system that aims to improve transparency, efficiency, and 
citizen engagement in municipal waste management by providing a centralized platform for reporting and tracking waste-related issues.

## Objectives
- Develop a centralized digital complaint management system
- Improve communication between citizens and municipal authorities
- Provide real-time complaint tracking
- Promote environmental responsibility

### User Module
- User Registration & Login
- Secure Authentication
- Submit Waste Complaints

- View Complaint History

- Update Complaint Status
